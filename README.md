### Data Analytics & Power BI Dashboard Project

This project presents a **market research study** focused on understanding user experience, adoption patterns, and improvement needs for the **FedMobile Mobile Banking Application** offered by Federal Bank Ltd. The dashboard visualizes how customers use the app, challenges faced, sentiment scores, and feature enhancement priorities.

## ğŸ” Key Findings
- Majority respondents heavily use **fund transfer** and **bill payment** features.
- **Ease-of-use** and **reliability** are the strongest drivers of user satisfaction.
- **Crash issues** and **linking errors** negatively impact trust and retention.
- There is high user interest in enhancing:
  - **Chat support**
  - **Investment & loan modules**
  - **Simplified UI navigation**
